# Discord-Akairo v0.0.1

A simple Visual Studio Code extension that allows you to quickly generate code for Discord-Akairo projects.

To install, click this [Link](https://marketplace.visualstudio.com/items?itemName=discord-akairo-snippets).

**This tool is still in development, if you would like to suggest any additions, please contact me at Swiizyy#2919!**

## Discord.JS snippets

| Name | Description |
| :---: | :---: |
| aka.index.template | Create a basic Discord bot index.js file. |
| aka.client.template | Create a basic client.js file |
| aka.commands.template | Create a basic listeners.js file |
| aka.listeners.template | Create a basic commands.js file |
